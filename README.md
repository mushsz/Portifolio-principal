## 🚀 Como Executar

1. **Clone o repositório**
   ```bash
   git clone https://github.com/mushsz/portfolio-matheus-krigger.git
   cd portfolio-matheus-krigger
   ```

2. **Instale as dependências**
   ```bash
   npm install
   ```

3. **Execute o servidor de desenvolvimento**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador**
   ```
   http://localhost:5174
   ```


### Design Responsivo
